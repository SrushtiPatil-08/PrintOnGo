# PrintOnGo 🚀

### Skip the Line. Print on Go.

PrintOnGo is a smart document printing and delivery platform that allows students, professionals, and businesses to upload documents, customize print settings, place orders online, and receive printed copies without visiting a print shop.

The platform transforms a traditionally time-consuming process into a seamless digital experience, making printing faster, more convenient, and accessible from anywhere.

---

## 📖 Problem Statement

Traditional printing services require users to:

* Travel to a print shop
* Wait in long queues
* Carry pen drives or email documents manually
* Explain print requirements repeatedly
* Spend valuable time collecting printed documents

For students and professionals with tight schedules, this process is inefficient and inconvenient.

---

## 💡 Our Solution

PrintOnGo digitizes the entire printing workflow.

Users can:

✅ Upload documents securely

✅ Customize print settings

✅ Get real-time pricing

✅ Place orders online

✅ Track their order status

✅ Receive printed documents through delivery

No queues. No travel. No wasted time.

## 🔄 How It Works

### 1. Upload Documents

Users upload their documents directly through the platform.

### 2. Customize Printing

Configure print preferences for each document individually:

* Black & White or Color
* Single or Double-Sided
* Stapling
* Spiral Binding
* Number of Copies

### 3. Instant Pricing

The platform automatically calculates the final cost based on:

* Number of Pages
* Print Type
* Color Selection
* Binding Options
* Quantity

### 4. Order Processing

Orders are routed to the nearest fulfillment center where documents are printed according to selected specifications.

### 5. Delivery

Printed documents are securely packed and delivered to:

* Home
* Office
* Campus Pickup Point

---

## 🎓 Batch-Hub Campus Delivery Model

PrintOnGo introduces a unique Batch-Hub delivery system specifically designed for students.

Instead of delivering every order individually:

* Student orders are consolidated
* Bulk deliveries are dispatched to campus pickup points
* Logistics costs are reduced
* Delivery becomes faster and more affordable

This model enables cost-efficient printing services for educational institutions.

## 🚀 Installation

### Prerequisites

* Node.js 18+
* npm or Bun

### Clone Repository

```bash
git clone https://github.com/SrushtiPatil-08/my-print-delivery.git
```

### Navigate to Project

```bash
cd my-print-delivery
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:8080
```
