# PrintOnGo

PrintOnGo is a modern document printing and delivery platform that allows users to upload files, calculate printing costs, select print preferences, and get documents delivered to their location.

## Overview

PrintOnGo aims to simplify printing by connecting customers with nearby print service providers through a seamless digital experience.

The platform is designed for:

* Students
* Professionals
* Offices
* Educational institutions
* Small businesses

## Features

* Document upload
* Automatic page detection
* Black & White and Color printing options
* Cost estimation
* Delivery request system
* Live location sharing support
* Mobile-friendly responsive interface
* Modern UI built with React and TypeScript

## Tech Stack

* React 19
* TypeScript
* Vite
* TanStack Start
* TanStack Router
* Tailwind CSS
* Radix UI
* React Query

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:8080
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
